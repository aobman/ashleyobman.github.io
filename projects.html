Interactive Application to Manage a Database

With Java and SQL programming languages, I have created a user-friendly, interactive application that allows one to manage a database. Given a table name, the user has the ability to retrive all data, insert new data, and delete data. If a column name is given in addition, the application will return the average of that column.

This application benefits the user by allowing them to recover, manipulate, and analyze data effectively while reducing complexity and boosting productivity. The use of dynamic querying saves time and effort when navigating large datasets. The statistical analysis functionality provides valuable insights into the dataset's numerical values, enabling informed decision-making.
